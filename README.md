# Todo App

A simple Todo application built with HTML, CSS and Vanilla JavaScript.

## Live Demo

🔗 https://mhdyrx.github.io/todo-app/

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using Local Storage

## Built With

- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned

- DOM Manipulation
- Event Delegation
- Local Storage
- Dynamic Rendering
- Object Manipulation
